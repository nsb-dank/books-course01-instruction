# 【演習】見掛けの傾斜の求め方を理解する  

フィールド調査を行い、下図のようなルートマップを作成しました。 ここで、礫岩と砂岩の境界が見られる露頭（境界面の走向・傾斜はN20E）を東西を切る（赤線）の地質断面図を作成しようと思います。

***  

【解説】  

（１） 断面図上の地層の見掛けの傾斜を計算しましょう。

```math
tan α = tan δ × sin θ で
θ = 70°、δ = 30°  であるから、
  tan α = 0.577 × 0.940 = 0.542
  α = 28.5°
```
